# AidNexus  -Acid Victim Management System

A database project developed for the **Advanced Database Management System (ADMS)** course.

The main goal of this project is to organize and manage information related to acid attack victims in one centralized database. The system keeps track of victim information, case records, medical treatment, financial support, legal documents, appointments, and notifications.



## Project Overview

Managing victim information through separate files or manual records can lead to missing data, delays, and poor tracking.

This project provides a structured database where every part of a victim's case can be stored and managed in one place.

The database follows proper normalization and relational design to keep the data organized and reduce duplication.
